# File Comparator Tool

Програма для порівняння двох текстових файлів з реалізованим CI-процесом.

## Функції
- Знаходження однакових рядків у двох файлах (`same.txt`).
- Знаходження унікальних рядків (симетрична різниця) (`diff.txt`).
- Повне покриття Unit-тестами.
- Автоматична перевірка PEP8 та запуск тестів через GitHub Actions.

## Запуск
1. Встановіть залежності: `pip install -r requirements.txt`
2. Підготуйте `file1.txt` та `file2.txt`.
3. Запустіть скрипт: `python comparator.py`

## Тестування
Запуск тестів та генерація HTML-звіту:
```bash
pytest --html=report.html --self-contained-html
